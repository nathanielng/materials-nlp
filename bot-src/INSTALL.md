# Bot Installation

## Prerequisites

1. Git
2. Python 3.6

## Installation

```bash
git clone git@github.com:nathanielng/science-bot.git
# or git clone https://github.com/nathanielng/science-bot.git
cd science-bot/bot-src
pip install -r requirements.txt
```

## Running the Bot

```bash
python3 science-bot.py
```
